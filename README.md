#### 安装教程

1.把项目文件夹发到服务器根目录下，并启动服务器
2. 打开文件夹
3. 在根目录下打开命令行窗口，运行：npm start 启动前端页面

#### 使用说明

1. 导入数据库keon.sql
2. 修改数据库配置：static\php\application\config
3. 基本用户:wonder wonde.123

# serve with hot reload at localhost:8080
